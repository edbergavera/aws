As Ubuntu cloud images are uploaded and registered on the
Amazon EC2 cloud,they are referred to as AMI (Amazon Machine Images).
Each AMI is a machine template from which you can instantiate new servers.

{{ form.instance_type }}

Micro Instance - 613 MB of memory, up to 2 ECUs
(for short periodic bursts), EBS storage only, 32-bit or 64-bit platform.
Small Instance - 1.7 GB of memory, 1 EC2 Compute
Unit (1 virtual core with 1 EC2 Compute Unit), 160 GB of local
instance storage, 32-bit or 64-bit platform.

{{ form.keypair }}

You use an Amazon EC2 key pair each time you launch an EC2
Linux/UNIX or Windows instance. The key pair ensures that
only you have access to the instance. Each EC2 key pair
includes a key pair name, a private key file, and a public key file.
